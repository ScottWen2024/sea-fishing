# 海钓通 产品需求说明书（PRD）

> 版本：v3.0 | 日期：2026-06-01

---

## 一、产品概述

### 1.1 一句话

**海钓通 = 出海决策 + 组队广播 + 渔获档案 + 海钓知识。** 让每一次出海都不靠猜。

### 1.2 我们解决什么问题

中国有 1.4 亿钓鱼爱好者，其中海钓群体超过 500 万人。但海钓市场存在三个"信息黑洞"：

**黑洞一：出海决策全靠猜。** 普通天气 App 只告诉你温度，不告诉你浪高、潮汐、适不适合出海。钓友每天打开三四个 App 拼信息，最后还得在微信群里问"今天能出吗"。

**黑洞二：组队靠微信群吼。** 拼船凑人、找靠谱船长、确认时间地点，全在微信群里刷屏。信息碎片化、无法沉淀、鸽子率高。

**黑洞三：渔获没有"家"。** 钓上来的鱼拍个照发朋友圈就完了，过两个月想不起来什么时候在哪钓了多大那条。渔获数据散落在相册和聊天记录里。

### 1.3 给谁用

海钓通的核心用户分四类，但最重要的两类是**新手**和**异地钓客**——这才是有痛点但没被服务到的人。

海钓通的核心用户分**两大类**，痛点截然不同：

#### 类型一：需要"领进门"的人

| 子类型 | 特征 | 核心痛点 |
|------|------|------|
| 🎓 新手钓客 | 0-3 次出海经验 | 不认识鱼、不会找钓点、不知道带什么、不敢一个人出海 |
| ✈️ 异地钓客 | 去新城市想钓鱼 | 对当地港口和船长一无所知，找不到人带 |

**共同需求：** 导钓服务、拍照识鱼、装备清单、安全须知、鱼种百科。他们需要有人"领进门"。

#### 类型二：需要"兜底"的人

| 子类型 | 特征 | 平时怎么做 | 什么时候需要 App |
|------|------|------|------|
| 🎣 休闲钓友 | 有固定钓友圈，偶尔缺人 | 朋友群里吼一声 | **朋友都没空时**——打开 App 发约钓广播，凑两个陌生的同路人 |
| 🏆 资深钓手 | 有固定船长，有好几个船家微信 | 直接联系老船家 | **熟悉的船家都没档期时**——打开 App 找其他认证船长的排期 |

**为什么老手也会用：**

```
平时不打开 ── 关系网够用时 App 没有存在感
     │
     ├── 熟悉的船家今天满了
     ├── 三个钓友这周都没空  
     ├── 到了一个新港口出差
     │
     ▼
打开海钓通 ── 5 分钟解决问题
     │
     ▼
问题解决，退出 ── 下次需要再用
```

老手不需要被"转化"，App 是他们的**保险**——平时用不上，但熟悉的渠道断了的时候，有个地方能兜底。这才是产品对老手的价值。

他们不可能把所有船家的排期都记住，也不可能认识每个港口的船长。当常联系的船长没档期，或者去新城市出差想顺便钓一次的时候，App 就是他们的 B 计划。

#### 老手的"救命"功能

哪些功能是老手平时不用、需要时能救命的：

| 场景 | 发生了什么 | 救命功能 | 体验 |
|------|------|------|------|
| 熟悉船家都满了 | 常联系的三个船长发微信说今天满了 | **找船长 + 排期查询** | 打开 App → 按港口筛选 → 看到谁的排期还有空位 → 直接预约 |
| 钓友群没人响应 | 发到群里半天没人回 | **约钓广播** | 打开 App → 发起约钓 → 所有打开 App 的人都能看到 → 有人加入 |
| 到陌生港口出差 | 到阳江出差，想顺便钓鱼，但这里一个船家都不认识 | **GPS + 港口选择 + 找船长** | 点 📡 定位 → 自动匹配最近港口 → 显示所有认证船长 |
| 不确定今天能不能出 | 看起来天气还行，但风浪不好判断 | **钓鱼指数 + 浪高条** | 首页一眼看到 6 维评分 → 红黄绿三种结果 |

核心原则：**不改变老手现有的关系网，只在关系网断裂的时候兜底。**

#### 用户画像全图

除了按资历划分，还有更多维度的人需要海钓通：

| 维度 | 角色 | 画像 | 核心需求 |
|------|------|------|------|
| **按经验** | 🎓 零基础新人 | 一次没去过，怕晕船、怕被坑、怕危险 | 安全须知、导钓推荐、装备清单 |
| | 🆕 去过几次 | 知道基本流程，但还没固定船家 | 找船、识鱼、约人拼船 |
| | 🏆 老资历 | 有固定船家和钓友圈 | 兜底：船家满了时找替代、陌生港口找船 |
| **按钓法** | 🦑 夜钓鱿鱼党 | 深圳大鹏夏季限定，晚上出动 | 夜间天气潮汐、鱿鱼钓点、装备推荐 |
| | 🪨 矶钓爱好者 | 站礁石上钓，对潮汐浪高极度敏感 | 精确潮汐数据、浪高预警、钉鞋救生衣 |
| | 🚤 船钓为主 | 坐船出海，深水钓大鱼 | 深水钓法、船长排期、大鱼排行 |
| | 🎣 路亚玩家 | 假饵诱鱼，追求操作技巧 | 路亚装备、钓点鱼种、标点记录 |
| **按身份** | 👧 女性钓手 | 怕没厕所、怕不安全、怕被歧视 | 女性友好船长标记、装备推荐、安全感 |
| | 👨‍👧 亲子出游 | 想带孩子体验钓鱼 | 安全优先的船长、温和行程、亲子半价 |
| | 🏢 企业团建 | HR 组织 8-20 人出海 | 一次找多条船、团体报价、正规发票 |
| | 📹 钓鱼自媒体 | 直播/拍视频，需要内容 | 爆护预测、热门钓点、渔获数据 |
| | 🌴 游客体验 | 来海边玩，从没钓过鱼 | 一次性的体验套餐、拍照留念服务 |
| | 👴 退休银发族 | 时间多但体力有限 | 近海短程、节奏舒缓的船长、不早起 |
| **按角色** | 👨‍✈️ 船长/船东 | 有船、有经验、需要客源 | 排期管理、获客工具、评价管理 |
| | 🎓 专业导钓 | 专门带新手，不一定自己有船 | 导钓服务展示、资质认证、学员管理 |
| | 🏪 渔具店老板 | 需要触达新客户 | 装备推荐中的导购链接、本地曝光 |
| | 🐟 渔获买手 | 餐厅/海鲜档收购渔获 | 渔获市场、批量采购、联系方式 |

**不是所有人都需要打开 App。** 船长和导钓用"我是船长"模式，游客和银发族可能只被家人推荐用一次，自媒体用渔获数据做内容。每个人只用他需要的那一部分。

### 1.4 为什么是我们

现有竞品做的是"工具"，我们做的是"全链路"：

| 场景 | 天气类 App | 船长撮合 App | 微信群 | **海钓通** |
|------|:---:|:---:|:---:|:---:|
| 今天能不能出海 | ✅ | ❌ | 靠经验 | ✅ **6 维钓鱼指数** |
| 潮汐怎么看 | 部分 | ❌ | ❌ | ✅ **潮汐曲线** |
| 找人组队 | ❌ | 部分 | ✅ 但碎片 | ✅ **约钓广播** |
| 找靠谱船长 | ❌ | ✅ | ❌ | ✅ **认证+评价** |
| 新手怎么开始 | ❌ | ❌ | ❌ | ✅ **导钓+装备+安全** |
| 渔获记录 | ❌ | ❌ | ❌ | ✅ **时间线+排行** |
| 钓多了怎么办 | ❌ | ❌ | ❌ | ✅ **渔获市场** |
| 这鱼能吃吗 | ❌ | ❌ | ❌ | ✅ **识鱼+百科** |

### 1.5 产品范围

**做：** 出海决策（天气/潮汐/指数）、组队（约钓广播/找船）、渔获（识鱼/记录/交易）、知识（百科/装备/安全）

**不做：** 支付（微信转账自己搞定）、物流（渔获自提/同城）、社交社区（MVP 阶段不做信息流）
