## 怎么生成这份报告

每次跑完测试，对 Claude 说：

> "跑测试并生成报告"

Claude 会：
1. 读取 `npm test` 的原始输出
2. 解析通过/失败数量
3. 填入这份模板
4. 保存为 `tests/reports/YYYY-MM-DD.md`

报告存档在 `tests/reports/` 目录下，方便追溯每次测试结果。
