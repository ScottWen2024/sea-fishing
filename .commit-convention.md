# Commit 消息规范 - 海钓通

## 格式

```
<type>(#<ticket-number>): <description>

<body>
```

## type（必选）

| type | 含义 |
|------|------|
| feat | 新功能 |
| fix | 修复 Bug |
| docs | 文档变更 |
| style | 格式（不影响逻辑） |
| refactor | 重构（不改变功能） |
| test | 测试相关 |
| chore | 构建/工具变更 |
| ci | CI 配置变更 |
| perf | 性能优化 |

## ticket 关联（强制）

- 所有 commit 必须关联 OpenProject ticket：`(#NN)`
- AI 完成功能后，通过 API 更新 ticket 状态

## 示例

```
feat(#15): 添加潮汐预测功能

- 集成和风天气潮汐API
- 新增 tide-predict 页面
- 添加潮汐时间轴组件
```

```
fix(#22): 修复渔获记录图片上传失败

原因：云存储路径未正确处理中文文件名
修复：使用 wx.cloud.uploadFile 的 cloudPath 参数编码
```
