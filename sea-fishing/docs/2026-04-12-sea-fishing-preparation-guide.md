## 一、海钓基础知识

### 1.1 海钓主要类型

| 类型 | 说明 | 适合人群 |
|------|------|---------|
| 船钓 | 搭乘专业海钓船出海垂钓 | 入门到进阶 |
| 拖钓 | 船尾拖放假饵或活饵 | 有经验者 |
| 矶钓 | 在礁石或码头垂钓 | 进阶玩家 |
| 滩钓 | 在沙滩/滩涂垂钓 | 入门玩家 |
| 筏钓 | 在渔筏上垂钓 | 有当地资源 |

### 1.2 主要对象鱼种

- **底层鱼**：石斑、红斑、鸡鱼
- **中层鱼**：鲈鱼、马鲅、海鲻
- **表层鱼**：鲔鱼、旗鱼、鬼头刀

## 二、船钓装备清单

### 2.1 基本装备

| 装备 | 推荐规格 | 必带/可选 |
|------|---------|-----------|
| 2米硬调海竿 | 或专用船钓竿 | 必带 |
| 大号卷线器 | 电动或旋压式 | 必带 |
| 钓鱼天平/连接器 | 船钓专用 | 必带 |
| 抄网及搭钩 | 抄网口大网深 | 必带 |
| 子线/钩/小配件 | 多规格备用 | 必带 |
| 钓饵 | 活虾/南极虾/管虫 | 必带 |

### 2.2 绕线轮类型

| 类型 | 优点 | 缺点 |
|------|------|------|
| 电动式 | 操作方便、自动收线 | 体积大、携带不便 |
| 旋压式 | 操作简单、收线快、体积适中 | 需要手动 |
| 密封式 | 防水性好 | 价格较高 |

### 2.3 装备保养

**鱼竿保养：**
1. 每节擦拭干净后晾干
2. 用汽车上光蜡擦拭杆体
3. 晾干后使用，光滑又减少磨损

**绕线轮保养：**
1. 拆下卸力器
2. 用200号汽油清洗
3. 每个连接部位用牙签沾白色润滑油润滑
4. 每年保养一次可延长寿命5倍

## 三、新手注意事项

### 3.1 安全须知

| 注意事项 | 说明 |
|---------|------|
| 穿救生衣 | 上船即穿 |
| 听从指挥 | 不擅自下水 |
| 关注天气 | 大风大雨不出海 |
| 告知行程 | 让家人知道去向 |
| 位置共享 | 开启手机定位 |

### 3.2 晕船应对

| 时机 | 建议 |
|------|------|
| 出行前 | 吃晕船药（提前30分钟） |
| 出海中 | 保持视线看远方/海平面 |
| 出海中 | 避免低头看手机 |
| 出海中 | 保持通风 |

### 3.3 防晒必备

| 物品 | 说明 |
|------|------|
| 防晒霜 | SPF50+，每2小时补涂 |
| 防晒衣 | 长袖、冰袖 |
| 墨镜 | 偏光镜最佳 |
| 帽子 | 宽檐帽 |

## 五、不同钓鱼方式的装备差异

### 5.1 船钓装备

```
- 80号至500号不等的船钓竿
- 6尺至15尺长度
- 手动或电动的船钓卷线器
- 船钓用编织布线或白铁线母线
- 船钓用倒吊或散尾钓组
- 船竿架、电瓶等
```

### 5.2 重矶钓装备

```
- 重矶竿或100号以上并继竿
- 母线至少80磅
- 子线至少100磅
- 铅负荷8两以上
- 挪威钩或石斑卧眠钩
- 鸭嘴固定器、铅块、钓组等
```

### 5.3 浮游矶钓装备

```
- 0号到2号矶钓竿
- 1000型到3000型卷线器
- 母线3号到6号浮水线
- 子线1号到5号碳纤线
- 太空豆、转环、水中浮标、夜光珠等
```

## 六、海钓六大海钓铁律

1. **安全第一** — 救生衣、天气判断、听从指挥
2. **装备适配** — 钓法与装备必须匹配
3. **钓点选择** — 潮汐、地形、鱼群分布
4. **饵料对路** — 不同鱼种用不同饵
5. **耐心守候** — 海钓需要时间
6. **环境保护** — 带走垃圾、放流幼鱼
